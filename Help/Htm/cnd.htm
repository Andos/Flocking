Flocking Object - Conditions

On looped boid
Parameters:
Loop name - The loop name
Object - Which object is looped

Is triggered one time for each of the boids that met the loop criteria when the loop was started.
Very useful for advanced control over the boids.

Note: A maximum of 9 nested loops can run at one time.
Note: This triggered condition must be the top condition of the event in which it resides.
