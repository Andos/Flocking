Getting boid-type attributes

Get idle speed
Expression:

IdleSpeed("Flocking", >boidtype<)

boidtype - The boid-type from which to get the attribute value

Returns: The idle speed attribute from the given boid type.

From object (via fixed value)

Get angle
Expression:

AngleFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The current angle of the object.

 

Get speed
Expression:

SpeedFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The current speed of the object (in pixels per time unit)

 

Get X-speed
Expression:

XSpeedFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The current X-speed of the object (in pixels per time unit)

 

Get Y-speed
Expression:

YSpeedFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The current Y-speed of the object (in pixels per time unit)

 

Get target X
Expression:

TargetXFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The X coordinate relative to the object where the object aims to move towards.

 

Get target Y
Expression:

TargetYFromFixed("Flocking", >fixed value<)

fixed value - The fixed value of the object from which to get the value

Returns: The Y coordinate relative to the object where the object aims to move towards.

 

From currently looped boid

Returns: The current angle of the currently looped object.
